# Voice Quest — 게임 기획서 (컨셉)

작성일: 2026-07-16 · 버전: 0.1 (MVP 단계)

## 1. 한 줄 컨셉

**"내 목소리가 곧 조작이자 대화가 되는, 사람 사는 것 같은 3인칭 소셜 월드"**

플레이어는 오버숄더 3인칭 시점으로 캐릭터의 뒷모습을 보며 세계를 돌아다닌다.
터치(클릭)와 **음성**으로 캐릭터를 조종하고, 명령이 아닌 말은 **그대로 채팅이 되어**
주변 사람들에게 말풍선으로 전달된다. 다른 플레이어들과 유대 관계를 쌓고,
함께 몹을 잡고, 운동회·중고시장 같은 생활형 콘텐츠에 참여한다.

## 2. 핵심 차별점 (Why this game?)

1. **음성 = 조작 + 대화의 통합**
   - "공격해", "멈춰" → 캐릭터가 즉시 수행 (명령)
   - "안녕하세요~ 같이 사냥하실래요?" → 그대로 채팅 말풍선 (대화)
   - 별도 모드 전환 없이 시스템이 문맥으로 구분한다.
2. **인식 실패가 캐릭터의 '연기'가 되는 UX**
   - 음성 인식이 실패하거나 모르는 명령이 들어오면, 에러 메시지 대신
     **캐릭터가 뒤돌아 플레이어를 바라보며 어깨를 으쓱하고 고개를 갸웃**한다.
   - 기술적 한계를 캐릭터의 성격/매력으로 전환 → 인식률이 완벽하지 않아도 즐겁다.
3. **"캐릭터 = 나" 몰입 설계**
   - 1인칭처럼 느껴지는 오버숄더 3인칭. 항상 캐릭터의 뒷모습을 보며 따라간다.
   - 캐릭터가 뒤돌아볼 때 카메라를 고정해 "쟤가 (화면 밖의) 나를 본다"는 느낌을 만든다.
4. **전투 게임이 아니라 '생활 월드'**
   - 몹 사냥은 콘텐츠 중 하나일 뿐. 목표는 광장에서 수다 떨고, 운동회에 참가하고,
     중고시장에서 흥정하는 "사람 생활" 경험.
5. **거친 말이 '미칭(美稱)'으로 바뀌는 세계 — "바른말 고운말" 컨셉**
   - 욕설·비하 표현은 말풍선에 그대로 나가지 않고 **"왕자님", "공자님", "대협"** 같은
     반대(존칭) 표현으로 치환되어 표시된다.
   - 차단·삭제 대신 무협 세계관의 예절로 승화하는 유머 — 갸웃 제스처와 같은
     "부정 경험을 캐릭터의 매력으로 바꾸는" 설계 철학의 연장.
   - 상세 컨셉과 교육적 효과는 §2-1 참조.

### 2-1. 바른말 고운말 — 교육·사회적 가치

이 세계에서는 나쁜 말이 존재할 수 없다. 뱉는 순간 고운 말로 승화된다.
단순한 욕설 필터가 아니라 **게임의 정체성이자 사회적 가치 제안**이다.

- **아이들에게: 안전한 첫 소셜 게임**
  - 욕설·비하가 화면에 노출되지 않는 채팅 환경 — 부모가 안심할 수 있는 멀티플레이.
  - 나쁜 말이 우스꽝스러운 존칭으로 바뀌는 경험 자체가 놀이가 되어,
    "바른말이 더 재미있다"는 습관을 자연스럽게 만든다.
- **성인에게: 바른말 전도사**
  - 홧김에 뱉은 거친 말이 "왕자님"으로 승화되는 순간, 웃음과 함께 자신의
    말버릇을 돌아보게 된다 — 처벌이 아니라 유머로 작동하는 자정 장치.
  - 커뮤니티 분위기가 거칠어질수록 오히려 존칭이 늘어나는 역설이
    게임 커뮤니티의 톤을 지킨다.
- **발음·발화 훈련 효과 (음성 조작의 부수 효과)**
  - 조작과 대화가 모두 음성 인식 기반이라, **또박또박 정확히 말해야
    캐릭터가 원하는 대로 반응**한다 — 플레이 자체가 발음 연습이 된다.
  - 인식 실패 시 갸웃 제스처가 "다시 또렷하게 말해볼까?"라는 부드러운
    피드백 루프가 되어, 아이의 언어 발달 연습과 성인의 명료한 발화
    습관(스피치 연습)에 도움을 주는 효과를 기대할 수 있다.
- 포지셔닝 한 줄: **"말이 고와지고, 발음이 또렷해지는 게임"**

## 3. 타겟 & 플랫폼

- 타겟: 캐주얼 소셜 게임 이용자. 채팅 타이핑이 불편한 모바일 유저,
  메타버스형 소셜 경험을 원하는 10~30대.
- 확장 타겟 (바른말 고운말 컨셉, §2-1): 안전한 채팅 환경을 원하는 학부모·아이 동반 가족,
  언어 습관·발음 개선에 관심 있는 성인 (교육·힐링 게임 포지셔닝 병행 가능).
- 플랫폼: 1차 웹(PC/모바일 브라우저, 설치 없음) → 2차 모바일 앱.
- 아트: 무협풍 카툰 스타일 (열혈강호 감성의 오리지널 디자인).
  ※ 실제 서비스 시 특정 IP와 혼동되지 않도록 독자적 캐릭터 디자인 필요.

## 4. 코어 루프

```
[접속/로비] → 닉네임 + 방 입장 (또는 월드 접속)
   ↓
[탐험] 터치/음성으로 이동 · 주변 플레이어 발견
   ↓
[소통] 음성이 채팅 말풍선으로 → 대화 → 유대감 형성
   ↓
[활동] 몹 협동 사냥 / 운동회 미니게임 / 중고시장 거래
   ↓
[보상] 유대감·아이템·칭호 → 관계가 깊어질수록 할 수 있는 활동 확장
   ↺
```

## 5. 유대(관계) 시스템 기획

- 플레이어 간 1:1 유대감 수치. 함께 한 활동으로 상승:
  - 근처에서 대화 주고받기 (+1~2)
  - 함께 몹 처치 (+4)
  - 미니게임 같은 팀 (+5), 거래 완료 (+3)
- 유대 단계: 낯선 사람 → 아는 사이 → 친구 → 절친 → 의형제(무협 컨셉)
- 단계별 해금: 원거리 귓속말, 아지트 초대, 합동 기술(협공 스킬), 전용 이모트.
- 설계 원칙: **수치를 보여주기보다 캐릭터의 행동 변화로 표현**
  (친해지면 캐릭터가 상대를 보고 먼저 손을 흔드는 등).

## 6. 생활 콘텐츠 기획 (로드맵)

### 6.1 운동회 (정기 이벤트형)
- 시간제 이벤트: 정각마다 운동장 존에서 개최, 방송으로 전체 알림.
- 종목(초기 3종):
  - **달리기**: 트랙 위 체크포인트 통과 레이스. 조작은 터치 연타 or "달려!" 음성 부스트.
  - **줄다리기**: 팀 나눠 타이밍 맞춰 "당겨!" 외치기 — 음성 볼륨/타이밍이 힘이 된다.
  - **보물찾기**: 맵에 숨겨진 오브젝트 찾기, 팀 채팅 협동.
- 보상: 칭호, 코스튬, 시장에서 쓸 재화.

### 6.2 중고시장 (상시 존)
- 시장 존에 좌판을 편다 → 아이템 등록(가격, 설명).
- 구매자가 좌판 앞에서 음성으로 흥정 → 판매자가 가격 수정/수락.
- 사기 방지: 시스템 에스크로(양쪽 확인 후 동시 교환).
- 재미 포인트: 시장 특유의 왁자지껄함 — 말풍선이 밀집되어 떠 있는 풍경 자체가 콘텐츠.

### 6.3 경제 시스템 — 벌기·모으기·쓰기 (소유의 재미)

생활 월드의 동력이 되는 인게임 경제 루프. 재화는 **은자(銀子, 가칭)**.

```
[벌기] 몹 드랍(요괴 가죽·정수) 수집 → 시장 판매
       내 물건 되팔기 (중고시장 §6.2 연계)
       다른 플레이어의 지원(후원) 받기
   ↓
[모으기] 은자 잔고 (HUD 표시) + 아이템 인벤토리
   ↓
[쓰기] 집: 부지 구매 → 건설 → 가구·마당 꾸미기
       탈것: 자전거·자동차 등 (이동 속도/연출 보너스)
       옷·외형: 도포/두건/검 스킨 등 코스튬
   ↓
[보여주기] 소유물이 월드에 그대로 노출 — 집들이 초대, 탈것 타고 광장 등장
   ↺
```

- **벌기의 3경로**
  - 사냥: 몹 처치 시 드랍 아이템 → 시장에 팔아 은자화. 전투가 경제로 이어진다.
  - 판매: 내 인벤토리 아이템을 중고시장 좌판에 등록 (§6.2의 에스크로 거래).
  - **후원(지원받기)**: 다른 플레이어가 은자를 보내줄 수 있다.
    유대 단계(§5) "친구" 이상만 가능 + 일일 한도로 악용 방지.
    "집 짓는 데 보태 써" — 후원 자체가 유대감을 올리는 소셜 행동이 된다.
- **쓰기 = 소셜에 보이는 소유**
  - 집(하우징): 이 경제의 최종 목표재. 부지 구매 → 단계별 건설 → 꾸미기.
    아지트 초대는 유대 단계 해금(§5)과 연결 — 집들이가 소셜 이벤트가 된다.
  - 탈것: 자전거·자동차 등. ※ 무협 세계관 조화를 위해 스킨을 각색할 수 있다
    (말·수레·축지 신발 등) — 기능(이동 속도·연출)은 동일하게 유지.
  - 옷·외형: 구매 즉시 캐릭터에 반영되어 광장에서 보인다.
- 설계 원칙: **은자는 게임 활동으로만 획득** — 현금 구매 배제(§7 P2W 배제 원칙과 일관).
  유료 상품은 외형 중심으로 별도 트랙 유지.

### 6.4 이후 후보
- 낚시터, 무공 수련장(대련), 장터 공연(악기/노래방).

## 7. 수익 모델 (초안)

- 코스튬/외형 (도포 색·머리 모양·두건·검 스킨)
- 아지트 꾸미기, 이모트/제스처 팩
- 배틀패스형 시즌 운동회 패스
- ※ P2W(전투력 판매) 배제 — 소셜 게임 신뢰의 핵심.

## 8. 리스크 & 대응

| 리스크 | 대응 |
|---|---|
| 음성 인식 지연/오인식 | 정밀 조작은 터치 담당, 음성은 명령/대화 담당(하이브리드). 실패는 갸웃 제스처로 UX화 |
| 음성 채팅 욕설/괴롭힘 | 욕설을 미칭(왕자님/공자님 등)으로 치환해 표시(구현됨, §2-5) → AI 모더레이션 고도화, 신고/차단, 존별 음소거 |
| 명령 vs 대화 오분류 | 명령조 어미 휴리스틱 → LLM 의도 분류로 고도화. 오분류 시 갸웃 제스처로 자연스럽게 회복 |
| 초기 동시접속 부족 | 소규모 방(P2P) 기반 친구 단위 성장 → 서버 월드로 확장 |
